# AI-Based Nutrition & Health Analyzer

## Project Overview

The AI-Based Nutrition & Health Analyzer is a full-stack web application designed to analyze food items, calculate nutritional values, and generate health scores based on food quality. The system helps users understand whether a food item is healthy, moderate, or unhealthy using machine learning classification techniques.

---

## Features

* Food nutrition analysis
* Health score calculation
* Food classification system
* Healthy / Moderate / Unhealthy prediction
* User-friendly web interface
* Real-time food data processing

---

* Machine Learning
* Pandas

## Installation

### Clone the Repository

```bash
git clone https://github.com/epmubashira325-bot/AI-Food-Nutrition-Analyzer
```

### Navigate to Project Folder

```bash
cd nutrition-health-analyzer
```

### Install Required Libraries

```bash
pip install -r requirements.txt
```

---

## How to Run the Project

```bash
python app.py
```

OR

```bash
streamlit run app.py
```

---

## Project Workflow

1. User enters food item details
2. System processes nutritional data
3. Machine learning model analyzes food quality
4. Health score is generated
5. Final classification is displayed

---

## Machine Learning Model

The project uses a classification model trained on nutritional datasets to predict:

* Healthy Food
* Moderate Food
* Unhealthy Food

Model Accuracy: **90%**
